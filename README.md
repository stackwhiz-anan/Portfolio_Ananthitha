# 🌐 Ananthitha V — Personal Portfolio

A clean, responsive personal portfolio website reflecting my skills, work, and growth as a Software Engineer.

🔗 **Live Demo:**(https://portfolio-ananthitha.vercel.app)

---

## 📌 Sections

- **Home** — Introduction and social links
- **About** — Brief bio, personal info and resume download
- **Skills** — Programming languages, web technologies, backend, databases and tools
- **Internships** — Real-world experience at Genz Educate Wing, Test Yantra, and Infomatronics
- **Projects** — Full-stack and Java projects with live demo and GitHub links
- **Certifications** — Industry certifications from NPTEL, Nasscom, British Council, Infosys and Hackerrank
- **Contact** — Contact form and social media links

---

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript

## ✨ Features

- Fully responsive design — works on all devices
- Smooth scroll animations and reveal effects
- Certificate viewer with image upload support
- Contact form with validation
- Hamburger menu for mobile navigation
- Downloadable resume
